# Git 學習筆記  
> 此內容教學以 Windows系統 為主。  
  
---
  
## 如何開始你的 Git 第一步  
- [Git 官網下載](https://git-scm.com/downloads)  
  
1. 下載完後，右鍵開啟 CMD命令提示單元/PowerShell  
  
2. 輸入 `git --version` 測試是否安裝成功，此時通常會顯示git版本代表已成功安裝。
